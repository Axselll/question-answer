## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/Axselll/qa

# Access
$ cd qa

# Install dependencies
$ go get

# Run the project
$ go run .

# The server will initialize in the <http://localhost:6969>
```
